Changing Images

To change any images of the website

  • Carefully collect the source name of the image (i.e. hero.jpg)
  • Open the file called img.
  • Find the particular image file.
  • Replace the file with your image.
  • Make sure that the file name does not change.The file name should be the same.

Changing Default Colors

We have used css for this project to write css. If you are familiar with css you should customize only the .css files. Please make sure you compile the main.css file after you have made changes in any of the .css files.

  • Open the custom-style.css file from assets > css > folder with a text-editor.
  • Change the right-side values of the variables to change any default colors of your site.
  • Save your file.
